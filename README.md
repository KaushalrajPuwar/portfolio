# Kaushalraj Puwar - Portfolio

A modern, interactive developer portfolio built to showcase my work, skills, and experience. Designed with a dark cosmic aesthetic featuring gold accents and a WebGL-powered interactive galaxy background.

## Features

- **Interactive Galaxy Background:** A custom WebGL star field that reacts to mouse movement for an immersive, modern feel.
- **Sectioned Layout:** Clean, single-page scroll navigation through Hero, About, Skills, Projects, Education, Publications, and Contact sections.
- **Project Showcase:** Filterable project cards with descriptions, tech stacks, and links to source code/live demos.
- **Responsive Design:** Fully fluid typography and layout that looks great on desktop, tablet, and mobile devices.
- **Smooth Animations:** Scroll-reveal animations and subtle hover effects implemented with Framer Motion.

## Tech Stack

This project is built using:
- **[React](https://react.dev/)** (with Vite for fast bundling)
- **[Tailwind CSS](https://tailwindcss.com/)** (for utility-first styling)
- **[Framer Motion](https://www.framer.com/motion/)** (for scroll and layout animations)
- **[OGL](https://github.com/oframe/ogl)** (for lightweight WebGL background rendering)
- **[Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)** (for scalable vector iconography)

## Aesthetics

The design system prioritizes visual excellence and a premium feel:
- **Palette:** Deep space black (`#0a0a0a`), warm cream (`#fdfbf7`), and striking gold/amber accents.
- **Typography:** Elegant serif headings paired with clean, readable sans-serif body text.
- **Glassmorphism:** Subtle translucent cards and overlays to maintain background visibility without sacrificing readability.
